Privacy Policy

Effective date: 14 July 2026

DocSendGo provides QR-based file sharing, document viewing, printing, downloading and document tools. This policy explains the information used to provide those features.

Information used

  • Profile details entered by the user, such as name, profile photo, mobile number, city and user type.
  • Files uploaded for sharing and related details such as file name, size, upload time, sender and receiver identifiers.
  • Technical information needed for app operation, security and error diagnosis.

How information is used

Information is used to create profiles, generate personal QR codes, deliver files to the selected receiver, show file activity and operate the requested tools.

Storage and service providers

DocSendGo uses Firebase services for database and file storage and Vercel for website delivery. Their systems may process data to provide these services.

User choices

Do not upload files you are not authorised to share. Users should remove files they no longer need where deletion controls are available. Keep personal QR codes and device access secure.

Data safety

No internet service can guarantee absolute security. DocSendGo uses the available platform security controls, but users should avoid sharing highly sensitive documents unless necessary.

Children

DocSendGo is a general productivity service and is not designed specifically for children.

Changes

This policy may be updated as the service changes. The latest version will be published on this page.
