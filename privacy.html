Privacy Policy

DocSendGo provides QR-based document sharing, file receiving, viewing, printing, downloading and selected document tools. This policy explains the data used to provide those features.

Information processed

  • Profile information voluntarily entered by a user, such as name, profile photo, mobile number, city and user type.
  • Uploaded files and related information such as file name, size, upload time, sender identifier, receiver identifier and activity status.
  • Technical information required for security, operation and error diagnosis, such as device/browser details and app version. Precise location is not required for the main file-sharing flow.

Permissions

  • Camera: requested only when the user starts QR scanning.
  • Files: used when the user chooses a document or image to upload, open or download.
  • Network: required to send, receive and synchronise files.

How data is used

Data is used to create profiles, generate QR codes, send files to the selected receiver, show activity, provide printing/download controls, prevent abuse and respond to support requests.

Service providers

DocSendGo uses Firebase services for database and file storage and Vercel for web delivery. These providers process information only as needed to provide the service under their own terms and safeguards.

Retention and deletion

Shared files and activity records may remain until they are removed by an available delete control, an administrator, a retention process or a valid deletion request. Users should not treat DocSendGo as permanent backup storage. To request deletion, use the instructions on the Contact page.

User responsibilities

Upload only files you own or are authorised to share. Confirm the receiver before sending. Do not upload illegal content or expose another person’s confidential information without permission.

Security

No online service can guarantee absolute security. Avoid uploading highly sensitive documents unless necessary, keep receive/share codes private and remove files when they are no longer needed.

Children

DocSendGo is a general productivity service and is not specifically directed to children.

Changes

This policy may be updated when features or legal requirements change. The latest version will be posted on this page.
