DocSendGo Multi-Document Update

এই আপডেটে:
- একটি file অথবা একসাথে একাধিক file নির্বাচন করা যাবে।
- প্রথমবার file নেওয়ার পরে “Add More Files” দিয়ে আরও file যোগ করা যাবে।
- প্রতিটি file আলাদাভাবে Remove করা যাবে।
- “Clear All” দিয়ে সব selected file সরানো যাবে।
- সর্বোচ্চ 20টি file একসাথে পাঠানো যাবে।
- প্রতিটি file সর্বোচ্চ 20 MB এবং মোট সর্বোচ্চ 100 MB।
- সব selected file-এর জন্য একটি Share Code ও একটি QR তৈরি হবে।
- Receiver একই code দিয়ে সব file View / Print / Download করতে পারবে।
- QR scan করে direct send করার ক্ষেত্রেও একই multi-file ব্যবস্থা রয়েছে।
- Service Worker cache version আপডেট করা হয়েছে।
